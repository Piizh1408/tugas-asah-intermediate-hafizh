                      JIKA PORT 8080 TERPAKAI
================================================================================

Jika muncul error "Port 8080 already in use", buka file:
starter-project-with-webpack/webpack.dev.js

Cari baris: port: 8080

Ganti dengan port lain, contoh:
port: 3000

Lalu jalankan lagi: npm run start-dev

Akses dengan: http://localhost:3000

================================================================================
                      ERROR YANG SERING MUNCUL
================================================================================

ERROR: "npm is not recognized"
SOLUSI: Install Node.js dari nodejs.org

ERROR: "Cannot find module"
SOLUSI: Jalankan: npm install

ERROR: "Port already in use"
SOLUSI: Ganti port di webpack.dev.js atau tutup aplikasi lain

ERROR: "Execution Policy"
SOLUSI: Buka PowerShell sebagai Administrator, ketik:
        Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned

ERROR: "Connection refused"
SOLUSI: Pastikan npm run start-dev masih berjalan di terminal
