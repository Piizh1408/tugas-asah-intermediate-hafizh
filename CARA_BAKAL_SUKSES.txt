LANGKAH-LANGKAH YANG PASTI BERHASIL:

1. BUKA FILE EXPLORER
   - Masuk ke: C:\Users\HP\Documents\tugas hafidz\starter-project-with-webpack

2. INSTALL DEPENDENCIES
   - Klik kanan di folder "starter-project-with-webpack"
   - Pilih "Open in Terminal" atau "Open in PowerShell"
   - Ketik: npm install
   - Tunggu sampai selesai (sekitar 1-2 menit)

3. BUILD PRODUCTION
   - Di terminal yang sama, ketik: npm run build
   - Tunggu sampai muncul "webpack compiled successfully"

4. JALANKAN
   - Ketik: npm run serve
   - Akan muncul pesan: "Available on http://localhost:8080"

5. BUKA DI BROWSER
   - Buka Chrome/Edge
   - Ketik di address bar: localhost:8080
   - Tekan Enter
