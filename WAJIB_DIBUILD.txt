================================================================================
              LANGKAH WAJIB: REBUILD & HARD REFRESH
================================================================================

STEP 1: REBUILD
---------------
Buka Terminal di folder starter-project-with-webpack, lalu:

npm run build

TUNGGU sampai muncul: "webpack compiled successfully"

STEP 2: SERVE
-------------
npm run serve

TUNGGU sampai muncul: "Available on http://localhost:8080"

STEP 3: BUKA BROWSER
--------------------
1. Buka: http://localhost:8080
2. PENTING: Tekan Ctrl+F5 untuk hard refresh (clear cache!)
3. Atau buka DevTools (F12) → Network tab → Centang "Disable cache"

STEP 4: TEST LAGI
-----------------
1. Login
2. Tambah cerita
3. Submit

================================================================================
              JIKA LOGGING TIDAK MUNCUL
================================================================================

Jika setelah rebuild dan hard refresh, console TIDAK menampilkan:
- "Response status: 400"
- "API Error Response:"

BERARTI: Browser masih pakai cache lama!

SOLUSI:
1. Tutup semua tab browser
2. Tutup browser
3. Buka browser lagi
4. Buka: localhost:8080
5. Test lagi

ATAU:

1. Buka DevTools (F12)
2. Klik kanan pada tombol refresh
3. Pilih "Empty Cache and Hard Reload"

================================================================================
              JIKA LOGGING MUNCUL
================================================================================

Jika console menampilkan "API Error Response: {message: "..."}"

Copy pesan error lengkapnya dan report!

Kemungkinan penyebab:
1. Format foto tidak didukung
2. Ukuran foto terlalu besar
3. Field tertentu tidak valid
4. API berubah requirement

================================================================================
              TROUBLESHOOTING
================================================================================

Q: "Response status: 400" muncul tapi tanpa "API Error Response:"
A: Server mengembalikan response bukan JSON
   Check Network tab → Response untuk lihat apa yang dikembalikan server

Q: Tidak ada log sama sekali
A: Browser masih pakai cache lama
   Clear cache lebih agresif (lihat di atas)

Q: Masih error setelah rebuild
A: Coba buka Incognito/Private window
   Test di Incognito dulu
